# 项目目录

```
├─admin
│  ├─build                     // vue-cli 配置
│  ├─config                    // vue-cli 配置文件
│  ├─dist                      // 打包后的文件
│  ├─src                       // 源代码
│  │  ├─api                      // 请求接口目录
│  │  ├─assets                   // 静态资源目录
│  │  ├─components               // 公共组件
│  │  ├─icons                  // 图标 svg
│  │  ├─mixins                 // mixin
│  │  ├─router                 // 路由文件
│  │  ├─store                  // vuex 状态管理
│  │  ├─styles                 // 公共样式
│  │  ├─utils                  // 公共方法
│  │  ├─views                  // 业务页面
│  │  ├─App.vue                // 主页面
│  │  ├─main.js                // 入口文件
│  │  ├─permission.js          // 监控路由跳转文件
│  ├─static                    // 部分静态资源CDN及图片
│  ├─test                      // 测试用例
│  ├─index.html                // 模板文件
│  ├─package.json              // 模块管理文件
```
